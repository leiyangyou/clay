What's Ralph Loop?

        

          
An autonomous coding loop. Claude writes code, a separate judge reviews the result, and if it's not done yet, Claude tries again — repeating until the judge says it passes.

          

            

              Claude codes
              →
              Judge reviews
              →
              Done
            

            

              
              Fail — retry
            

          

          
In the next step you'll describe what to build. Claude will draft the instructions, and you can preview and edit them before anything runs.

          

        

      

      

        
What do you want to build?

        
Write a rough idea — Clay will refine it into detailed instructions. You can review and edit everything before the loop starts.
